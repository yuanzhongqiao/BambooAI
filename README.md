<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">BambooAI 代理通过几个关键步骤与用户交互并生成响应：</font></font></p>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">1. 启动</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">用户带着问题启动 BambooAI 代理。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果未提供初始问题，代理将提示用户提出问题或“退出”命令以终止程序。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">然后，代理进入一个循环，对提供的每个问题做出响应，并在完成后提示用户下一个问题。</font><font style="vertical-align: inherit;">这个循环一直持续到用户选择退出程序为止。</font></font></li>
</ul>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">2. 任务评估</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">代理存储收到的问题并利用大语言模型（LLM）对其进行评估和分类。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">法学硕士确定问题是否需要文本回复、附加信息（谷歌搜索），或者可以使用代码解决。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">根据任务评估和分类，代理调用适当的代理。</font></font></li>
</ul>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">3. 动态提示构建</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果问题可以通过代码解决，代理将确定所需的数据是否包含在所提供的数据集中、是否需要从外部源下载，或者问题是否具有通用性质并且不需要数据。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">然后代理相应地选择其方法。</font><font style="vertical-align: inherit;">它制定了一种算法，以任务列表的形式表示，作为分析的蓝图。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">修改原始问题以与该算法保持一致。</font><font style="vertical-align: inherit;">该代理针对向量数据库执行语义搜索以查找类似问题。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">找到的任何匹配问题都会作为示例附加到提示中。</font><font style="vertical-align: inherit;">然后使用 GPT-3.5、GPT-4 或本地 OSS 模型根据算法生成代码。</font></font></li>
</ul>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">4. 调试、执行和纠错</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果生成的代码需要调试，则使用 GPT-4。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">代码被执行，如果发生错误，代理会记录错误消息并将其返回给 LLM 进行更正。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">此过程将持续到成功执行代码为止。</font></font></li>
</ul>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">5. 结果、排名和知识库构建</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">成功执行后，GPT-4 用于对答案进行排名。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果排名超过设定的阈值，问题、答案、代码和排名将存储在 Pinecone 矢量数据库中。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">无论排名如何，最终答案或可视化都会被格式化并呈现给用户。</font></font></li>
</ul>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">6. 人类反馈和循环延续</font></font></strong></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">代理寻求用户的反馈。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果用户验证自动生成的排名，则问题/答案对将存储在矢量数据库中。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果没有，则开始新的执行循环。</font></font></li>
</ul>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在整个过程中，代理不断征求用户输入，存储上下文消息，并生成和执行代码以确保最佳结果。</font><font style="vertical-align: inherit;">在此过程中采用了各种人工智能模型和矢量数据库，以对用户的问题提供准确且有用的答复。</font></font></p>

This method is particularly effective for vague user prompts, but it might not perform as efficiently with more specific prompts. The default setting is True.
